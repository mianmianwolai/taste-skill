# Taste-Skill UI 设计说明

## 设计系统

### 色彩体系

**主色**：蓝色 (#1890FF)
- 用途：主要按钮、链接、高亮元素

**辅助色**：
- 成功绿 (#52C41A)
- 警告黄 (#FAAD14)
- 错误红 (#FF4D4F)
- 中性灰 (#BFBFBF)

### 字体
- 中文：阿里巴巴普通话字体 / 微软雅黑
- 英文/数字：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto

### 组件库
使用 Element Plus

## 响应式设计

### 屏幕尺寸
- PC: ≥ 1200px
- 平板: 768-1200px
- 手机: < 768px

### 移动端优化
- 字体大小 ≥ 16px
- 按钮高度 ≥ 44px
- 触摸区域 ≥ 44x44px
- 表格简化为卡片式
